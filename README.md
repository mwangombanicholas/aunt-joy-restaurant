# Aunt Joy's Restaurant - Online Ordering System

A complete web-based food ordering system built with PHP, MySQL, HTML, CSS, and JavaScript.

## Features
- Customer ordering interface
- Admin meal management
- Sales order processing
- Manager reporting dashboard
- Role-based access control

## Installation
1. Install XAMPP
2. Place files in htdocs folder
3. Import database.sql to phpMyAdmin
4. Access via http://localhost/aunt-joy-restaurant
